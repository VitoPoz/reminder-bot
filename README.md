# Reminder Bot

Telegram-бот-напоминалка на aiogram с асинхронной обработкой таймеров.

## Возможности

- Установка напоминания текстовой командой в свободной форме
- Асинхронное ожидание срока без блокировки бота для других пользователей
- Поддержка минут и часов в запросе

## Стек

Python, aiogram, python-dotenv, asyncio

## Установка и запуск

```bash
git clone https://github.com/VitoPoz/reminder-bot.git
cd reminder-bot
python -m venv venv
venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Создай файл `.env` в корне проекта и укажи токен бота, полученный через @BotFather:
```bash
BOT_TOKEN=твой_токен
```

Запусти бота:
```bash
python bot.py
```

## Использование

Напиши боту `/start`, затем в формате:
```bash
напомни через 10 минут купить хлеб
напомни через 1 час позвонить клиенту
```

## Планы по развитию

- Переход с разбора свободного текста на inline-кнопки/FSM (более надёжный ввод)
- Хранение напоминаний в базе данных (сейчас таймеры живут только в памяти процесса — при перезапуске бота все активные напоминания теряются)
- Поддержка отмены напоминания
